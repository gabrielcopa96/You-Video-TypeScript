## Running the client

```
# install dependencies
npm install

# run in dev mode on port 3001
npm start
```

## Running the api

```
# install dependencies
npm install

# run in dev mode on port 3001
npm run dev

# generate production build
npm run build

# run generated content in dist folder on port 3001
npm run start
```

## Testing the api

### Jest with supertest

```
npm run test
```
